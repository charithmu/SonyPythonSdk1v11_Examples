Image ID String Property

The Image ID String property can be retrieved and set using the get_property(‘ImageID_String’) and set_property(‘ImageID_String’, ‘Off’) methods respectively on a connected camera.

In the Camera Remote SDK, the API call to retrieve the property will be GetDeviceProperties .

In the Camera Remote SDK, the API call to set the property will be SetDeviceProperty.

import sony_cr
import io
import sys
import time

cm = sony_cr.CameraManager()

cameras = cm.find_cameras()

print("Found {} camera(s)".format(len(cameras)))
for i, cam in enumerate(cameras):
    print("[{}]: {} {}".format(i+1, cam['model'], cam['id']))

if len(cameras) == 0:
    sys.exit(0)

cam = cm.get_camera(cameras[0]['id'])

cam.connect()

cam.set_property('ImageID_String', 'test')

response = cam.get_property('ImageID_String')

print(response)

Description

Once the camera library has been initialised and you have connected to a camera (see Initialise the Camera Library), you can start issuing commands, setting or getting parameters.

Call the get_property() method with a string to get the required property (‘ImageID_String’ in this case).

Return Value

An example of the object returned from this call is below.

{'possible': ['Off', 'On'], 'readable': True, 'value': 'Off', 'writable': True}

Property Details can be found in the Camera Remote SDK.

Running the Sample

Copy the code snippet above into a file called ImageID_String.py and execute using the command:

python ImageID_String.py

Properties

See Samples/Properties for usage examples.

Here is the list of the currently supported properties for the python library. For the full list of properties, please see function list.

Please note that when the table provides the link to the possible parameters of properties, the ‘CrDeviceProperty_’ or ‘Cr[parameter-name]_’ prefixes should not be used. It been removed from the code for the sake of simplicity.

Description
